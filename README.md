# IA para Médicos — Landing Page

Landing page do curso **"IA para Médicos: Inteligência Artificial aplicada à prática clínica e gestão"**, programa de educação continuada do Hospital Israelita Albert Einstein.

## Sobre o projeto

Esta página foi construída inteiramente com **Claude Code** como showcase de como é possível criar uma landing page profissional usando IA em pouquíssimo tempo.

**Stack:** HTML, CSS e JavaScript puros — sem frameworks, sem build tools.

## Como rodar localmente

Abra `index.html` no navegador. Não precisa de servidor.

## Estrutura

```
├── index.html          # Página principal
├── css/styles.css      # Estilos (design system, responsivo)
├── js/main.js          # Animações (partículas, scroll reveal, carrossel)
└── assets/images/      # Imagens e logos de ferramentas
```
